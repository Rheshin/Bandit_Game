# Bandit Writeup — OverTheWire

Writeup personnel du wargame [Bandit](https://overthewire.org/wargames/bandit/) 
proposé par OverTheWire. Chaque niveau est documenté avec la solution et une 
explication des concepts abordés.

## Format de chaque niveau

- **Objectif** : ce qu'on cherche à faire
- **Commandes utilisées** : avec explication
- **Solution** : démarche pas à pas
- **Concepts clés** : ce qu'on apprend

## Avertissement

Les mots de passe ne sont pas inclus, l'objectif est de comprendre 
la démarche, pas de copier les réponses.
